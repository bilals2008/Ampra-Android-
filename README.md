UI Framework: Jetpack Compose (Declarative UI) with Material 3 dynamic standard templates.
Architecture: MVVM (Model-View-ViewModel) with structured Flow states.
Database: Room Database for local persistence of charging sessions & customized profiles.
Themes: 8 Beautiful themes (Light, Dark, AMOLED, Cyberpunk, Solar Amber, Forest Moss, and Retro Lavender).
Core Services: Background BatteryMonitorService for real-time tracking, custom vibration types, blink camera flashlight alerts, and TTS (Text-to-Speech) triggers.
