# Technical Foundation

- UI Framework: Jetpack Compose (Declarative UI) with Material 3 dynamic standard templates
- Architecture: MVVM (Model-View-ViewModel) with structured Flow states
- Database: Room Database for local persistence of charging sessions & customized profiles
- Themes:
  - Light
  - Dark
  - AMOLED
  - Cyberpunk
  - Solar Amber
  - Forest Moss
  - Retro Lavender
- Core Services:
  - Background `BatteryMonitorService` for real-time battery tracking
  - Custom vibration patterns
  - Blink camera flashlight alerts
  - TTS (Text-to-Speech) trigger supportUI Framework: Jetpack Compose (Declarative UI) with Material 3 dynamic standard templates.
